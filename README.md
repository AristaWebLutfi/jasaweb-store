# Jasaweb_Store

Website company profile untuk jasa pembuatan **Website, Game, dan Desain Grafis/Branding**. Dibangun sebagai proyek pengembangan web dengan tampilan responsif dan fitur interaktif.

🔗 **Demo:** `https://username-kamu.github.io/nama-repo/` <!-- ganti sesuai link GitHub Pages kamu -->

---

## 📄 Halaman

| Halaman | Deskripsi |
|---|---|
| `index.html` | Beranda — hero slider, layanan singkat, paket harga |
| `layanan.html` | Detail 3 layanan (Web, Game, Desain) + alur kerja |
| `kontak.html` | Peta lokasi, info kontak, form pesan (kirim otomatis via EmailJS) |
| `tentang_kami.html` | Profil, visi-misi, teknologi, tim, portofolio |
| `faq.html` | Pertanyaan yang sering ditanyakan (accordion) |

---

## 🛠️ Teknologi yang Digunakan

- **HTML5** — struktur halaman
- **CSS3** — styling & animasi (flexbox, grid, transition)
- **JavaScript (Vanilla)** — interaktivitas (slider, accordion, form)
- **Font Awesome** — ikon
- **EmailJS** — pengiriman pesan dari form kontak
